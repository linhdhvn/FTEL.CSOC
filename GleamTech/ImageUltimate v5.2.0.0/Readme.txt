ImageUltimate v5.2.0.0
ASP.NET Image Resizer
Copyright C 2015-2022 GleamTech
https://www.gleamtech.com/imageultimate

ImageUltimate is the fastest and easiest ASP.NET image resizer which supports ASP.NET Core 2.0+, ASP.NET MVC 3.0+ and ASP.NET WebForms 4.6.1+ web applications/web sites.
ImageUltimate can also be used with .NET Core 2.0+ and .NET Framework 4.6.1+ console/desktop applications for conversion between several image formats.

------------------------------------------------------------------------------------------------------
Information on package contents:
------------------------------------------------------------------------------------------------------

  - "Bin" folder contains the DLLs for ImageUltimate.
    - "Bin\net461" - These DLLs should be used for ASP.NET MVC 3.0+ and ASP.NET WebForms 4.6.1+ web applications/web sites
                    and also for .NET Framework 4.6.1+ console/desktop applications.
    - "Bin\netstandard2.0" - These DLLs should be used for ASP.NET Core 2.0+ web applications
                             and also for .NET Core 2.0+ console/desktop applications.
    Please see below for the full instructions on how to reference and use the DLLs in your ASP.NET project.

  - "Documentation" folder contains the API reference as a CHM file.

  - "Examples" folder contains the example projects for
    - ASP.NET Core (C#) - Visual Studio 2019+
    - ASP.NET Core on .NET Framework (C#)- Visual Studio 2017+
    - ASP.NET MVC (C#) - Visual Studio 2012+
    - ASP.NET MVC (VB) - Visual Studio 2012+
    - ASP.NET WebForms (C#) - Visual Studio 2012+
    - ASP.NET WebForms (VB) - Visual Studio 2012+
    Please open one of the solution files with Visual Studio 2022/2019/2017/2015/2013/2012 to load an example project. 
    The example projects demonstrate various features and settings of ImageUltimate components. Note that, 
    the projects reference GleamTech.ImageUltimate.dll which is at "Bin" folder of the root of this package 
    so make sure you extract the whole package (not only "Examples" folder) before opening a solution.

  - "Legal" folder contains the EULA for ImageUltimate.
